# ButtonRippleEffect
Button Ripple Effect A simple and interactive button ripple effect built using HTML, CSS, and JavaScript. The ripple originates from the click position, creating a smooth and modern UI feedback animation.

## Features

- Dynamic ripple effect on button click
- Ripple starts from cursor position
- Smooth CSS animation
- Lightweight and beginner-friendly
- No external libraries used

## Technologies Used

- HTML5
- CSS3 (Animations)
- JavaScript (DOM Manipulation)

##  How It Works

1. When the button is clicked, a `<span>` element is dynamically created.
2. JavaScript calculates the cursor position relative to the button.
3. The ripple is positioned at the click point.
4. CSS animation scales the ripple outward.
5. The ripple element is removed after the animation ends.
